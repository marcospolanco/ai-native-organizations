Typography

Font Hierarchy

  • Headings: -apple-system, 600 weight, 1.125x line height
  • Body: -apple-system, 400 weight, 1.5x line height
  • Code: Monaco/Courier, 400 weight, 1.4x line height
  • Messages: 16px size, natural spacing
  • UI Elements: 14px size, consistent padding

Spacing System

8px Base Unit:
Small elements: 4px, 8px
Component spacing: 12px, 16px
Section spacing: 24px, 32px
Page spacing: 48px, 64px

Interactive States

Button States

Hover

Input States

Animation & Transitions

  • Duration: 150ms for UI transitions, 300ms for content changes
  • Easing: ease-out for natural feel
  • Streaming: Smooth text appearance with cursor
  • Collapsible: Height transitions for panels
  • Loading: Subtle spinner animations

Responsive Breakpoints

  • Mobile: < 768px (single column, compact UI)
  • Tablet: 768px - 1024px (medium layout)
  • Desktop: > 1024px (full layout with max width)

Accessibility Guidelines

  • Color Contrast: WCAG AA compliant (4.5:1 minimum)
  • Focus States: Visible 2px outline for keyboard navigation
  • Screen Readers: Proper ARIA labels and semantic HTML
  • Touch Targets: Minimum 44px for interactive elements
  • Keyboard: Full navigation support, Enter to submit

Design Implementation Notes

  • Uses Tailwind CSS for utility-first styling
  • Radix UI components for accessibility foundation
  • Dark mode support via CSS custom properties
  • Responsive design with mobile-first approach
  • Consistent spacing using 8px grid system
